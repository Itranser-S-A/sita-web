¿Cómo funciona?

Tres pasos y listo para jugar.

1

Crea tu gymkana

Define el mapa, añade retos y personaliza reglas y tiempos. Puedes empezar desde una plantilla.

2

Comparte el código

Invita a tus jugadores con un enlace o código QR. Ellos se unen en segundos.

3

Juega y compite

Los equipos superan retos, suben pruebas y el ranking se actualiza en tiempo real.

Preguntas frecuentes

¿Sita es gratis?

Sí, puedes crear y jugar gymkanas gratuitas. Más adelante ofreceremos planes Pro para eventos grandes y colegios.

¿Necesito conexión a Internet?

Sita funciona offline para jugar; sólo necesitas conexión al crear o sincronizar resultados.

¿En qué dispositivos está disponible?

iOS y Android (disponible próximamente). También habrá panel web para organizadores.
